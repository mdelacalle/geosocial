geosocial by mdelacalle

GeoClient

GeoSocial is a tool for analysing and visualising spatial data from some of the most popular social networks, including Twitter, Facebook and many others. The application features an intuitive client connected to several social network's API's, which retrieves geolocated information from them and allows the user to analyse and explore it, offering a large array of ESDA routines and spatial analysis algorithms. These algorithms are provided by the SEXTANTE geoprocessing library, while NASA WorldWind is used for visualization.

Dependencies

Nasa World Wind, Sextante
